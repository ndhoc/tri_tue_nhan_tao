# Folder bài tập (btvn)

Đây là thư mục tổng hợp các bài tập/đồ án theo từng buổi học.

## Cấu trúc thư mục

- `buoi3/`, `buoi4/`, `...`, `buoi13/`: mỗi buổi thường có 1 `README.md` (hoặc file hướng dẫn) mô tả bài làm và cách chạy/chạy demo.

## Cách sử dụng

1. Chọn thư mục của buổi bạn quan tâm (ví dụ: `btvn/buoi8/`).
2. Đọc `README.md` trong thư mục đó để biết:
   - Bài toán/đề tài là gì
   - Các thuật toán/cách làm chính
   - Cách chạy chương trình

## Gợi ý đóng góp

- Nếu bạn thêm bài/đồ án mới cho một buổi, hãy cập nhật hoặc thêm `README.md` trong thư mục buổi đó.
- Nếu có thay đổi cách chạy, nhớ ghi rõ lệnh chạy (ví dụ: `python ...`).

Chúc bạn học tốt!
