# Bài tập buổi 3 - Robot hút bụi

Mô phỏng tác tử robot hút bụi phản xạ đơn giản (Simple Reflex Agent) trong không gian lưới 5x5.

## File code
- `24162039_NguyenDucHoc_Robot_Hut_Bui.ipynb`: Jupyter Notebook chứa code mô phỏng.

## Chức năng
- Tạo môi trường 5x5 ngẫu nhiên (ô sạch, ô có rác, vật cản).
- Robot tự động di chuyển ngẫu nhiên đến các ô kề cạnh.
- Nếu phát hiện rác, thực hiện hành động hút bụi.
- Ghi log lại các bước đi của robot.
