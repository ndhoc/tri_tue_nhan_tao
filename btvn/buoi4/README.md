# Bài tập buổi 4 - Robot hút bụi (Model-based Agent)

Mô phỏng tác tử robot hút bụi có trạng thái (Model-based Reflex Agent) trong không gian lưới 3x3.

## File code
- `24162039_NguyenDucHoc_May_hut_bui_model.ipynb`: Jupyter Notebook chứa code mô phỏng.

## Chức năng
- Khởi tạo không gian 3x3 với các ô có rác hoặc sạch.
- Tác tử ghi nhớ lại các bước đã đi (model/history).
- Di chuyển thông minh hơn dựa trên lịch sử để tránh đi lại các ô cũ nếu có thể.
- Hiển thị lịch sử di chuyển và trạng thái cuối cùng của môi trường.
