# Bài tập buổi 6 - Mô phỏng 8-Puzzle (BFS, DFS, IDS)

Ứng dụng giao diện đồ họa (GUI) mô phỏng các thuật toán tìm kiếm không có thông tin giải quyết bài toán 8-Puzzle.

## File code
- `24162039_NguyenDucHoc_Visualize_8Puzzle.py`: Mã nguồn Python (Tkinter).

## Thuật toán tích hợp
1. **BFS Version 1**: Tìm kiếm theo chiều rộng (phiên bản cơ bản).
2. **BFS Version 2**: Tìm kiếm theo chiều rộng (tối ưu tập frontier/explored).
3. **DFS**: Tìm kiếm theo chiều sâu.
4. **IDS**: Tìm kiếm sâu dần (Iterative Deepening Search).

## Chức năng giao diện
- Sinh ma trận ngẫu nhiên (đảm bảo có thể giải được).
- Chạy thuật toán tìm kiếm và báo cáo số node đã duyệt.
- Xem lại từng bước giải (chạy từng bước hoặc tự động).
- Hiển thị log chi tiết hướng di chuyển (L, R, U, D).
